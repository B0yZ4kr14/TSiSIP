# TSiSIP Memory Index

This is the durable memory index for the TSiSIP project. It is maintained by the
`memory-md` Spec Kit extension and updated via explicit human approval.

## Navigation

| File | Purpose |
|---|---|
| [PROJECT_CONTEXT.md](PROJECT_CONTEXT.md) | Product identity, domain language, key constraints |
| [ARCHITECTURE.md](ARCHITECTURE.md) | System shape, module boundaries, integrations |
| [DECISIONS.md](DECISIONS.md) | Architecture Decision Records (ADRs) and rejected patterns |
| [BUGS.md](BUGS.md) | Known bugs, root causes, and resolutions |
| [WORKLOG.md](WORKLOG.md) | Session history and completed work |
| [memory-synthesis.md](memory-synthesis.md) | Condensed context for agent planning |

## How to Update

1. Run `speckit.memory-md.capture` after completing significant work.
2. Review the proposed changes.
3. Approve or edit before committing.
4. Never commit secrets, PII, or environment-specific values.

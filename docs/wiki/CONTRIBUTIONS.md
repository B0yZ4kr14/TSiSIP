# TSiSIP Contributions

## How to Contribute

### Code
1. Fork repository
2. Create branch
3. Make changes
4. Submit PR

### Documentation
1. Find gap
2. Write content
3. Submit PR
4. Review

### Testing
1. Find bug
2. Write test
3. Fix bug
4. Submit PR

### Feedback
1. Use product
2. Share thoughts
3. Report issues
4. Suggest features

## Guidelines

### Code Style
- PSR-12 for PHP
- Standard for JS
- BEM for CSS

### Commits
- Conventional format
- Clear messages
- Atomic changes

### Tests
- All changes tested
- Integration preferred
- Coverage maintained

## Recognition

### Contributors
- Listed in CONTRIBUTORS.md
- Mentioned in releases
- Thanked in docs

### Rewards
- Swag
- Credits
- Recognition

# Changelog

All notable changes to the Work IQ Spec Kit Extension will be documented in this file.

## [1.0.0] - 2025-07-14

### Added

- `/speckit.workiq.ask` — Direct natural-language query to M365 data (emails, meetings, docs, Teams)
- `/speckit.workiq.context` — Gather M365 context across multiple data sources for spec writing
- `/speckit.workiq.stakeholders` — Discover decision-makers, SMEs, and contributors from M365 activity
- `/speckit.workiq.enrich` — Analyze and enrich existing specs with M365 organizational knowledge
- Workflow hooks for `before_specify` and `after_specify` integration
- README with installation, prerequisites, and usage documentation
- Detailed usage guide with examples (`docs/usage.md`)
- Configuration template (`config-template.yml`)

# TSiSIP Glossary

## A

**Asterisk**: Open-source PBX software used as backend.

**Audit Log**: Record of all user actions for compliance.

## B

**Bookmark**: Saved page for quick access.

## C

**CSRF**: Cross-Site Request Forgery protection.

**Control Panel (OCP)**: Web interface for managing TSiSIP.

## D

**Dark Mode**: Low-light color theme.

**Dispatcher**: OpenSIPS module for load balancing.

**Docker**: Container platform for deployment.

## E

**Edge Proxy**: Public-facing SIP proxy.

## G

**Gateway**: SIP endpoint for routing.

## H

**HA1**: Hash used for SIP digest authentication.

**Health Check**: Endpoint to verify system status.

## I

**i18n**: Internationalization (multi-language support).

**MI**: Management Interface for OpenSIPS.

## L

**Locale**: Language and region setting.

## M

**MI HTTP**: HTTP-based management interface.

## O

**OpenSIPS**: Open-source SIP server.

## P

**PBX**: Private Branch Exchange (phone system).

**PostgreSQL**: Relational database.

## R

**RTP**: Real-time Transport Protocol for media.

**RTPengine**: Media proxy for RTP.

## S

**SIP**: Session Initiation Protocol.

**SSE**: Server-Sent Events for real-time updates.

**Subscriber**: SIP user account.

## T

**Theme**: Visual appearance (light/dark).

**Topology Hiding**: Hiding backend network structure.

**TSiSIP**: The project name.

## U

**UI**: User Interface.

## W

**WebSocket**: Bidirectional communication protocol.

# Changelog

All notable changes to spec-kit-markitdown will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-23

### Added

- Initial release
- `/speckit.markitdown.convert` command — convert a single document to Markdown
- Support for PDF, Word, PowerPoint, Excel (core) and HTML, CSV, JSON, XML, Images, Audio, EPub, ZIP (extended)
- Automatic output to `.specify/specs/` directory
- Source metadata headers on converted files
- Output quality checks with suggestions for Azure Document Intelligence and OCR plugin

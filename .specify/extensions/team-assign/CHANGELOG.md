# Changelog

## [1.0.0] - 2026-05-16

### Added
- `/speckit.team.setup` command — interactive team member configuration
- `/speckit.team.assign` command — role-based, round-robin, and manual assignment strategies
- Subtask splitting for complex tasks with user confirmation
- Capacity tracking per engineer with over-capacity warnings
- `/speckit.team.board` command — per-engineer Markdown workboard
- `team.yml` configuration template
- Support for re-running assignment (update-only or full reassign)

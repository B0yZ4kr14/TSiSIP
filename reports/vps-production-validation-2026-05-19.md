## Issue Resolution

### Problem
- OpenSIPS container stuck in restart loop (exit 255)
- Logs showed unsupported method error for TLS range syntax

### Fix Applied
Changed TLS method from range syntax (TLSv1.2-TLSv1.3) to single version (TLSv1_2)

### Root Cause
OpenSIPS 3.6 tls_mgm module requires:
- Parameter name: tls_method
- Format: TLSv1_2 (underscore, single version)
